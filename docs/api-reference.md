# API Reference

## Planned Endpoints

- `POST /api/auth/register` - User registration
- `POST /api/auth/login` - Login
- `GET /api/plans` - List tracking plans
- `POST /api/plans` - Create tracking plan
- `GET /api/plans/:id` - Get plan details
- `PUT /api/plans/:id` - Update plan
